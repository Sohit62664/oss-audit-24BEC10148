## 📖 Project Description
This project is an audit and analysis of the Python programming language as an open-source software. It explores its origin, licensing model, ecosystem, and role in modern computing. 

Along with the report, five Linux shell scripts are developed to demonstrate practical understanding of open-source tools and command-line operations.

---

## ⚙️ Scripts Overview

### 1. Script 1 — System Identity Report
Displays system information including:
- Linux distribution
- Kernel version
- Current user
- System uptime
- Date and time

---

### 2. Script 2 — FOSS Package Inspector
- Checks if Python is installed
- Displays version, maintainer, and description
- Uses case statement for philosophy note

---

### 3. Script 3 — Disk and Permission Auditor
- Audits important directories
- Shows permissions, owner, and size
- Checks Python binary location

---

### 4. Script 4 — Log File Analyzer
- Reads log file line-by-line
- Counts occurrences of a keyword (default: error)
- Displays last 5 matching lines

---

### 5. Script 5 — Open Source Manifesto Generator
- Takes user input interactively
- Generates a personalized open-source philosophy
- Saves output to a text file
